Projects

    National Parks Dashboard

    With the love to explore nature, this project explores the unique features, species diversity and visitor trends of all 63 U.S. National Parks. Park information was scraped from Wikipedia using Python’s BeautifulSoup, cleaned with Power Query and Python and visualized in Power BI. The final interactive dashboard provides insights and acts as a recommendation engine, helping users discover their ideal destinations based on activities, biodiversity and park characteristics.

    H1B Trend Analysis

    Analyzed H-1B petition data from USCIS (2020–2024) using Python for ETL and Tableau for dashboards. Highlighted patterns in approvals, denials, top sponsors, and job roles. This project involved the comprehensive analysis of H-1B nonimmigrant worker petition data from USCIS.gov, covering fiscal years 2020 through 2024. Utilizing Python for data cleaning and preparation, the project aimed to uncover significant trends related to H-1B applications, approvals, denials, and employer sponsorship patterns. Key insights were then visualized and presented using Tableau, offering an interactive exploration of the data's various facets, including state-level distributions, top sponsoring companies and dominant industry codes.

    Food Delivery Time Prediction

    The food delivery time prediction model plays a crucial role in the food delivery industry. Delivery time directly impacts customer satisfaction and influences their overall experience. Using regression algorithms like linear regression, decision tree, random forest, XGBoost we built the predictive model. It was trained on a subset of the dataset using techniques like cross-validation to ensure robustness. Evaluation metrics such as mean squared error (MSE) and R-squared (R2) score were used to assess the model's accuracy. This model empowers businesses to optimize their operations and improve the overall delivery experience for their customers.

    Healthcare Data Breach NLP

    This project is about data breaches in healthcare threaten patient privacy and system integrity. With the rise of digital records, the risk of exposing sensitive information like medical histories and insurance details increases. These breaches can lead to identity theft, financial loss, and diminished trust in healthcare providers. This project aims to provide proactive breach risk management by leveraging text analytics, offering a powerful approach to analyze the data breaches pattern and characteristics from unstructured textual data. Text mining techniques such as logistic regression, topic modeling using advanced NLP methods like BERT, LDA, NER can uncover hidden patterns and anomalies, helping understand breaches in protecting sensitive data.

    Book Buying Behavior Model

    A case study analyzing customer purchasing behavior for a newly released book using KNN modeling and RFM analysis. The RFM (Recency, Frequency, Monetary) is a segmentation process in database marketing that aims to partition customers in a list of prospects into homogeneous groups (segments) that are similar with respect to buying behavior. This approach enabled the book owner to predict customer actions, optimize marketing spend and maximize ROI, demonstrating how data-driven strategies can tackle business challenges and drive impactful results.

    Netflix Movie Exploration

    Designed a robust data management system to improve data accessibility, analysis, and recommendation accuracy using insights from Netflix’s dataset. Built a star schema in Erwin Data Modeler, performed data warehousing in SQL Server, and utilized SSIS for ETL (Extract, Transform, Load) processes. Final insights were visualized in Tableau, enabling more effective recommendation algorithms and strategic decision-making in the movie industry.

    California COVID Analysis

    In this project, I analyzed the impact of COVID-19 across California demographics using Python (pandas, seaborn and matplotlib library) and statistical visualizations, focusing on age, gender and ethnicity trends in case and death rates. Extended the study to examine the correlation between population size and COVID-19 rates across US states and counties. I also leveraged publicly available datasets from the New York Times and federal sources to explore whether larger populations are associated with higher infection rates. This analysis offers valuable insights for public health policy, resource distribution and pandemic response strategies.
